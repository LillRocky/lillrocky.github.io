Python学习笔记--字典

发表于 2018-05-17 | 分类于 Python | 热度： ℃

字典

在Python中，字典是一系列键-值对。类似于JS中的对象。

增删改查

增加

dict[‘key’] = value

删除

del dict[‘key’]

修改

dict[‘key’] = new_value

查询

dict[‘key’]

遍历字典

遍历字典中的键值对

方法items()返回一个键-值对的列表

仅遍历字典中的键

方法keys返回一个键的列表

按顺序遍历字典中所有的键

对keys()方法返回的列表进行排序

遍历字典中所有的值

方法values返回一个值的列表

字典与列表的嵌套

有时候，为了复杂的需求，需要将一系列字典存储中列表中，或者将列表作为值存储中字典中，称为嵌套。
同时，字典也可以嵌套在字典中，列表也可以嵌套中列表中。
